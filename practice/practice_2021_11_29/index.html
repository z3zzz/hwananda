About this project

1. Why this project

프로젝트 아이디어 동기 NLP, 언어 데이터에 대한 관심 사회적 현에 대한 관심 감염자 수 대비 기사제목의 긍정/부정 감정 변화 정부 대책(거리두기 단계)에 따른 기사제목의 긍정/부정 감정 변화 질본의 발표내용에 따른 기사제목의 긍정/부정 감정 변화 문제를 해결하기 위한 특정 질문 명시 코로나 감염자 수의 변화가 뉴스 기사 제목에 미치는 영향은? 데이터를 통해 탐색하려는 문제를 구체적으로 작성 코로나 감염자 수와 뉴스 기사 감정 사이의 연관성을 수치화 할 수 있다. 언론사별/키워드별/날짜별 기사 제목의 긍정/부정/중립 비율을 시각화하여 볼 수 있다.

2. How this project

사용하려는 데이터 네이버에 노출되는 전체 기사 코로나 확진자 수(http://ncov.mohw.go.kr/) 설명 : 기사 제목, 언론사, 링크, 확진자 수를 크롤링해서 사용 크롤링 대상 기간 : 2020.01.01.~ 현재 기술 스택 프론트엔드 : HTML, CSS, Javascript 백엔드 : Django, Flask 데이터 분석 : PyTorch, sklearn DB : mongoDB 크롤링 : Selenium 사용된 라이브러리 공통 : numpy, matplotlib, pandas, pymongo, tqdm, re 프론트엔드 : chart.js, bootstrap, d3.js, pyLDAvis, wordcloud, konlpy 데이터 분석 :gluonnlp, seaborn, transformers, logging 크롤링 : requests, bs4, multiprocessing, platform 웹서비스에 대한 자세한 개요 사용자가 원하는 분류 별(일별, 키워드별, 언론사별)로 뉴스 기사 제목의 감정 분석 결과를 시각화(파이 그래프, 꺾은선 그래프, 표)하여 보여준다.
